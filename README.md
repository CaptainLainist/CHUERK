# CHUERK

A simple file shredder for windows

![alt text](https://raw.githubusercontent.com/CaptainLainist/CHUERK/main/chuerk.png)

# Instalation

Download the software in releases, unzip it and add it to path

# Tutorial
```bash
#normal shred
chuerk file

#to show help
chuerk -h

#recursive shred
chuerk -r folder
```
